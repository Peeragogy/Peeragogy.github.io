How to contribute: Micro Quickstart Guide

  1. Create Github Account https://help.github.com/articles/signing-up-for-a-new-github-account/
  2. (optional) Contact the Peeragogy Project in Gitter https://gitter.im/Peeragogy
  3. (optional) Create a https://hypothes.is/ account if you want to make comments.
  4. (optional) Connect with +Peeragogy Project on Google+ and join our weekly meetings.
  5. Start editing, discussing, and commenting.
Please read this longer quickstart guide for more information.
